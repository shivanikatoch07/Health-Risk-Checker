# Personalised Health-Risk-Checker
This is a project using simple Python based application that evaluates a user's health risk level bases on: 
-Age
-Blood pressure
-Fasting sugar level 
-Lifestyle
This project is made using conditional statements to classify risk as:
-Low Health Risk
-Medium Health Risk
-High Health Risk
# Features
-Simple user inputs
-Users if/elif/else decision making 
-Real-world health application
-Begineer-friendly Python projectt
-Outputs clear health assessment
#Technologies Used
-Python (Basic Input/Output)
-Conditional Statments
#How to Run
python health_risk_cheacker.py
